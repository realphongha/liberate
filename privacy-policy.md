# Privacy Policy for Translate Anyway

**Last updated: June 2026**

Translate Anyway does not collect, store, transmit, or share any personal data or browsing information.

## Data Collection

This extension collects **no data**. Specifically:

- **No browsing history**: We do not track or record the websites you visit.
- **No page content**: All translation-blocking modifications happen locally in your browser. No page content is sent to any server.
- **No analytics**: We do not use analytics, telemetry, or crash reporting tools.
- **No cookies**: We do not set, read, or transmit cookies.
- **No user accounts**: No registration or login is required.

## Data Storage

The only data stored is your extension settings (enable/disable state and site-specific preferences). This data is stored locally using `chrome.storage.sync` and is never transmitted to any third party. It is used solely to maintain your preferences across browser sessions and synced Chrome installations.

## Permissions

- **storage**: Used solely to save your preferences (enable/disable toggle, per-site settings).
- **activeTab**: Used solely to display the current website domain in the extension popup. No tab data is read, stored, or transmitted.

## Third Parties

This extension does not load any external resources, make network requests, or communicate with any third-party services.

## Changes

If this policy changes, users will be notified via the extension's update notes on the Chrome Web Store.

## Contact

For questions about this privacy policy, please open an issue on the extension's GitHub repository.
